Summary: `agent-browser` es una herramienta de Vercel que permite a los agentes de IA navegar por la web y realizar acciones como lo haría un humano, integrándose fácilmente con frameworks de agentes.

`agent-browser` (https://github.com/vercel-labs/agent-browser) me permite que los agentes de IA puedan navegar por la web y realizar acciones complejas como si fueran un humano. 

Lo tengo integrado en una skill y lo uso habitualmente con `pi agent` para automatizar tareas que requieren interacción con el navegador. Es brutal lo que se puede conseguir cuando el agente puede "ver" y "actuar" en la web directamente.
