Title: Colores desde web developer tools

Últimamente uso tanto [firebug](http://getfirebug.com/) que ni me acuerdo de
las [Web Developer Tools](https://addons.mozilla.org/en-US/firefox/addon/60).
Escribo esto para recordar que sigue siendo muy útil esta extensión. Por
ejemplo, puedo ver los colores (css) usados en una página web a través del
menú _Información_, seleccionando _Ver información del color_.
