Title: Cambiar ipython para copiar y pegar luego como doctest

Otra de las ordenes que nunca recuerdo. Estando dentro de
[ipython](http://ipython.scipy.org/moin/) para cambiar el prompt de forma que
luego sea fácil crear [doctests](http://docs.python.org/library/doctest.html):
**`%doctest_mode`** Esto entra en  el modo doctest de ipython. También puede
ayudar **`%history -tn`**` para recordar los commandos introducidos
previamente. `
