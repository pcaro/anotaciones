Title: Para aprender javascript

Para aprender una tecnología todos conocemos [nuestro punto de
referencia.](http://www.pablocaro.es/www.google.com) Pero "El Buscador" no es
punto de partida más recomendable para aprender javascript. Existen cientos,
miles, millones, de tutoriales donde aprender mal javascript. No hablo de
aprender las palabras clave y las sentencias de control. Para eso te vale
cualquiera. Hablo de aprender los conceptos clave detrás del lenguaje y su
implicación explicada de forma práctica. Para no perder el tiempo y comprender
de verdad el lenguaje incomprendido hay que acudir a:

  * [Mastering JavaScript — concept and resource guide](http://blog.reindel.com/2007/10/16/mastering-javascript-concept-and-resource-guide/) gracias a Brian Reindel
  * Las presentaciones de [Douglas Crockford](http://javascript.crockford.com/). Los vídeos están recopilados [en este enlace](http://101out.com/jss.php)
