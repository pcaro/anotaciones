Velvet OS: kernels modernos para hardware ARM antiguo

Publicado el 17 de April de 2026 en Miscellaneous

Nota mental sobre Velvet OS, un proyecto que descubrí a través de esta entrada en Raspberry Pi para Torpes.

Velvet OS es un framework sencillo para construir imágenes de tarjetas SD arrancables con Debian o Ubuntu para diversos dispositivos ARM. El objetivo es proporcionar kernels de Linux modernos para hardware que ya no tiene soporte oficial o que quedó abandonado.

Lo interesante del proyecto:

  • Soporta más de 30 sistemas diferentes, desde Chromebooks ARM hasta SBCs como Odroid y OrangePi
  • Kernels actualizados (actualmente 6.12) para hardware que se quedó estancado en kernels 3.x o 4.x
  • Imágenes listas para usar, sin necesidad de compilar nada
  • Proyecto mantenido activamente por hexdump0815

El nombre "Velvet OS" viene de que el autor suele cubrir las tapas rayadas de sus Chromebooks con terciopelo autoadhesivo, haciendo desaparecer el branding comercial. La idea es que usar estas imágenes sea una experiencia tan agradable como tocar terciopelo.

Tengo varios cacharritos antiguos por casa, incluyendo dos OrangePi. A ver si saco un ratito para actualizarlos con kernels modernos y darles una segunda vida.

El tutorial de Raspberry Pi para Torpes explica en detalle cómo actualizar una ODROID-U3 de 2013 al kernel 6.12, incluyendo los pasos para configurar el U-Boot, copiar el kernel, generar el initrd y ajustar el boot.ini. Todo el proceso está muy bien documentado.
