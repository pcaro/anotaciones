Rufus, la mejor forma de instalar un Windows limpio

Publicado el 27 de April de 2026 en Software

Leo en informática sin límites que Rufus 4.14 (en fase beta) mejora sustancialmente su capacidad para instalar un Windows limpio.

Rufus lleva años siendo la referencia para crear medios de instalación de Windows. La nueva versión va más allá:

  • Deshabilita Teams, Outlook, Copilot y otros programas innecesarios de Microsoft
  • Permite usar cuentas locales en lugar de obligar a usar cuentas Microsoft
  • Elimina BitLocker (el cifrado que trae de cabeza a usuarios y empresas)
  • Nuevo modo desatendido: automatización completa de la instalación sin intervención del usuario

Si necesitas un Windows sin toda la basura que Microsoft intenta imponer, Rufus es la herramienta.
